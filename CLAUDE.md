# oxy-workflow

## Agent skills

### Issue tracker

Issues 以本地 markdown 文件跟踪于 `.scratch/<feature>/`；无外部
PR 触达面。见 `docs/agents/issue-tracker.md`。

### Triage labels

五个 triage 角色均使用规范默认名（needs-triage / needs-info /
ready-for-agent / ready-for-human / wontfix）。见
`docs/agents/triage-labels.md`。

### Domain docs

单上下文布局：根目录 `CONTEXT.md` + `docs/adr/`。见
`docs/agents/domain.md`。
